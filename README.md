# 📚 OpenClaw 知识库

统一的技能和配置经验记录仓库。

## 📂 目录结构

```
knowledge_base/
├── README.md                    # 本说明文件
├── experience.md               # 📖 经验总结（统一记录，实时更新）
└── cron-backups/               # ⏰ 定时任务备份
    ├── README.md              # 使用说明
    └── latest.md              # 最新完整配置
```

## 📝 文件说明

### experience.md - 经验总结（主文件）

**用途**: 统一记录所有技能使用经验、配置方法、故障解决方案

**更新时机**:
- ✅ 掌握新技能后
- ✅ 解决重要问题后
- ✅ 配置成功新功能后
- ✅ 发现最佳实践后

**格式示例**:
```markdown
## 技能名称/主题

### 功能说明
简要描述这是什么功能

### 使用方法
具体步骤和命令

### 经验教训
遇到的问题和解决方案

### 更新时间: YYYY-MM-DD
```

### cron-backups/latest.md - 定时任务配置

**用途**: 存放所有定时任务的最新配置

**更新时机**: 每次添加/修改/删除定时任务时立即更新

## 🔄 推送机制

不再自动每日推送，改为**手动更新 + 关键节点推送**:

1. **我记录**: 遇到重要经验时，我主动更新 `experience.md`
2. **您指示**: 您说"更新知识库"时，我整理并推送
3. **关键节点**: 重大配置完成后推送备份
