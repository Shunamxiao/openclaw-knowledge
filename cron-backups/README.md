# ⏰ 定时任务备份目录

此目录存放 OpenClaw 所有定时任务的配置备份。

## 🎯 使用说明

- **latest.md** - 查看最新完整配置（总是最新的）
- **历史备份** - 带时间戳的备份（如 `2026-02-04_1043.md`）

## 🔄 更新时机

以下情况会更新此目录：

1. ✅ 添加新定时任务
2. ✅ 修改定时任务配置
3. ✅ 删除定时任务
4. ⚠️ 服务重启导致任务丢失时（紧急备份）

## 🚨 恢复定时任务

如果服务重启导致定时任务丢失：

### 方法1: 查看 latest.md
打开 `latest.md` 文件，复制对应的恢复命令执行。

### 方法2: 告诉我
直接说："恢复定时任务"，我会根据最新备份自动重建。
